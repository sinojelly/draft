## 🔧 技术实现亮点

### 1. 零依赖架构
- 纯原生 JavaScript（ES6+）
- 不使用任何框架或库
- 所有功能手写实现

### 2. 模块化设计
- ES6 模块系统（import/export）
- 组件化开发（每个页面/组件独立文件）
- 清晰的目录结构

### 3. 设计模式
- **MVC 模式**：App 作为控制器，Pages 作为视图
- **发布订阅**：EventBus 实现组件间通信
- **单例模式**：Toast、EventBus 全局唯一
- **策略模式**：不同页面实现相同接口

### 4. 代码质量
- 命名规范清晰
- 函数职责单一
- 注释完整
- 易于维护和扩展

## ⭐ 与 React 版本对比

| 特性 | Vanilla SPA | React 版本 |
|------|-------------|------------|
| 依赖 | 零依赖 | React + ReactDOM |
| 打包 | 无需打包 | 需要 Webpack/Vite |
| 体积 | ~50KB | ~150KB+ |
| 性能 | 极致优化 | 虚拟 DOM 开销 |
| 学习曲线 | 需要深入理解 DOM | 熟悉 React 即可 |
| 浏览器兼容 | 需支持 ES6 Module | 需支持 ES6 |
| 开发体验 | 手动管理 DOM | 声明式开发 |
| 适用场景 | 追求极致性能 | 快速开发 |
